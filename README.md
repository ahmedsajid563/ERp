# Media Mosiac ERP System v1.0

## Modules
- Dashboard (live stats, charts)
- Invoices (GST / Export / Proforma, line items, live PDF preview)
- Clients
- Client Payments + TDS tracking
- Expenses (category-wise, bulk export)
- Payroll + Salary Slips
- Password Vault (AES-256 encrypted)
- Bulk Export (CSV / JSON / PDF) with date filters

## Tech Stack
| Layer    | Technology                          |
|----------|-------------------------------------|
| Frontend | React 18 + Vite                     |
| Backend  | Node.js + Express                   |
| Database | MySQL                               |
| Auth     | JWT + bcrypt                        |
| Vault    | AES-256 (crypto-js)                 |
| Hosting  | Hostinger Business Web Hosting      |
